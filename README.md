# Jason Albano — Senior Data Analyst Portfolio Website

A modern, responsive static portfolio website for GitHub Pages.

## Files

- `index.html` — website content and structure
- `styles.css` — visual design, layout, responsiveness, animations
- `script.js` — mobile menu, scroll reveal animation, cursor glow

## How to run locally

Open `index.html` in your browser.

## How to publish on GitHub Pages

1. Create a new GitHub repository named `jason-portfolio` or `yourusername.github.io`.
2. Upload `index.html`, `styles.css`, `script.js`, and `README.md` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then click **Save**.
6. GitHub will publish the site at the Pages URL shown on that screen.

## Customize

Replace the LinkedIn placeholder URL in `index.html` with your real LinkedIn profile URL.
